微信参考说明
1：微信网页授权
https://mp.weixin.qq.com/wiki?t=resource/res_main&id=mp1421140842&token=&lang=zh_CN
涉及到的类
	WebAuthorizeRequest 网页授权请求
	WebAuthorizeAccessTokenRequest 通过code获取网页授权token
	WebAuthorizeAccessTokenResponse 通过code获取授权token的响应结果
	WebRefreshTokenRequest 刷新accessToken
	WebRefreshTokenResponse 刷新accessToken后的响应
	WeixinUserInfoRequest 拉取用户信息
	WeixinUserInfoResponse 用户信息响应结果
	WebAccessTokenInvalidRequest 验证token是否有效
